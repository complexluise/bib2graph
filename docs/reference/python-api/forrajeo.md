# Forrajeo y curación

Expandir y curar el corpus: el `Forager` rankea candidatos por *information
scent* (estructura bibliométrica determinista, sin IA), con preview de
crecimiento antes de comprometer; el `Preprocessor` normaliza, y los filtros
PRISMA acotan el corpus de forma trazable.
