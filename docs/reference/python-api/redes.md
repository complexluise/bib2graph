# Redes (proyección)

Proyectar el corpus a **redes bibliométricas**. `Networks` orquesta los cinco
proyectores —acoplamiento bibliográfico, co-citación, co-autoría, colaboración
institucional y co-ocurrencia de keywords—, cada uno produciendo un
`NetworkArtifact` listo para analizar o exportar.
